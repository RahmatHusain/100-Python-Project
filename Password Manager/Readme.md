![Status](https://img.shields.io/badge/Status-Completed-success)

## 📖 Overview

Managing passwords manually can be difficult and insecure. This Password Manager allows users to securely store and retrieve login credentials using a simple command-line interface.

The project demonstrates Python fundamentals such as file handling, JSON data storage, functions, error handling, and menu-driven applications.

---

## ✨ Features

- 🔑 Save website credentials
- 👀 Retrieve saved passwords
- 💾 Store data using JSON
- 🛡️ Error handling for invalid data
- 📂 Persistent storage
- 🚀 Beginner-friendly interface
