> Organize your messy folders automatically with Python.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Status](https://img.shields.io/badge/Status-Completed-success)

## 📖 Overview

**Automatic File Organizer** is a Python automation tool that scans a folder and automatically sorts files into categorized folders based on their file extensions.

Instead of manually organizing hundreds of downloaded files, this script completes the task in seconds.

---

## ✨ Features

- 📁 Automatically organizes files
- 🖼️ Supports image files
- 📄 Organizes documents
- 🎵 Moves audio files
- 🎬 Sorts video files
- 🚀 Fast and lightweight
- 🛡️ Skips existing folders safely
- 💻 Beginner-friendly Python project

---

## 🗂️ Supported File Types

| Category | Extensions |
|----------|------------|
| 🖼️ Images | .jpg .jpeg .png .gif |
| 📄 Documents | .pdf .doc .docx .txt |
| 🎬 Videos | .mp4 .avi .mkv |
| 🎵 Audio | .mp3 .wav |
